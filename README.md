# WPM-DOCUMENTACAO-RECEP

Repositório oficial da documentação do projeto **WPM Gestão Interna**.

## Finalidade
Este repositório concentra a documentação oficial do sistema, incluindo:
- documentação técnica
- contexto de versões
- auditorias
- registros de fechamento
- materiais de apoio do projeto

## Site publicado
A documentação visual publicada via GitHub Pages está neste próprio repositório, a partir do arquivo `index.html`.

## Relação com os demais repositórios
- `WPM-SISTEMA-RECEP` → sistema principal oficial
- `WPM-DOCUMENTACAO-RECEP` → documentação oficial
- `WPM-RECEP-CASE` → case oficial
- `WPM-RECEP-CASE-FRST` → legado / staging temporário

## Observação
Este repositório é destinado à documentação oficial do projeto e deve ser mantido separado do sistema principal.
